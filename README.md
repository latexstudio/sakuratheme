Sakura theme
============

This is a beamer template based on [Metropolis](https://github.com/matze/mtheme).
This can be used in languages which are written in non-latin characters.

## Typesetting

```
latexmk demo.tex
```

## Reference
+ [Metropolis](https://github.com/matze/mtheme)
